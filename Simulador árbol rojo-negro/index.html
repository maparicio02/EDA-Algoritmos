Visualizador de Árbol Rojo-Negro

Propiedades del Árbol Rojo-Negro

1. Cada nodo es rojo o negro.

2. La raíz siempre es de color negro.

3. Todos los nodos hojas son negros.

4. Los nodos rojos no pueden tener hijos rojos.

5. Cada camino desde la raiz a sus nodos descendientes null tiene el mismo número de nodos negros.
